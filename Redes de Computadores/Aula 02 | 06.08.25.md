# Endereço IP
Classes de redes:
* A - 1 à 127 bits
  * Começa com 0
* B - 128 à 191 bits
  * Começa com 10
* C - 192 a 223 bits
  * Começa com 110
* D - 224 a 239 bits
  * Começa com 1110
* E - 240 a 255
  * Começa com 1111
* Exemplo
  * **192**.168.1.5 - É de 192 à 233 = Classe C
  * **110**00000.10101000.00000001.00000101 - Começa com 110 = Classe C
