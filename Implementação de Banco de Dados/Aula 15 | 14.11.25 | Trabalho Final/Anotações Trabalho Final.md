# Informações sobre o trabalho
* Esse trabalho vale como a nota integral da terceira nota do semestre

# Situação Problema
* O Museu Treze de Maio necessita de soluções tecnológicas para gerenciar dois acervos distintos, cada um com suas especificidades
  * Acervo histórico
  * Biblioteca Comunitária

# Objetivo Geral
* Projetar, desenvolver e implementar o gerenciamento e a consulta publica do acervos do Museu

# Descrição da arefa
* Desenvolver um banaco de dados responsável
